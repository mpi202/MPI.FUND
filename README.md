# MPI.FUND

Website for **Marketprog Asset Management Zrt.** — a Hungary-based, MNB-regulated asset manager (AIFMD).

## About

Single-page static site with client-side routing, interactive charts (Chart.js), and real fund performance data. No build step required — just open `index.html` in a browser.

## Stack

- HTML + Tailwind CSS (CDN)
- Chart.js for interactive visualizations
- Inter font (Google Fonts)
- No frameworks, no Node.js required

## Deploy

Static hosting — drop the files on Vercel, Netlify, or any web server.

## Data

Fund YTD returns sourced from [Hozam Plaza](https://hozamplaza.hu/alapkezelok/261/marketprog) (32 funds, 17 families).
